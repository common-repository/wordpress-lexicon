== Installation ==

1. Download and extract lexikon.zip File
2. Upload `lexikon.php` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Settings - Lexikon and insert Page ID from the Page you want to activate it
5. Create New Page and change the Attribute  to Lexikon. All Lexicon entrys Pages add to Lexikon
6. Thats it. A german Videotutorial you find on my website
